Name
A name for this device, since MAC addresses are hard to remember.
MAC Address
You can get the MAC address by running:
  • Windows: ipconfig, displayed as "Physical Address."
  • OS X or Linux: ifconfig, displayed as "HWaddr."
Optional
The following fields are only needed if you are planning on waking up your device from a network other than the one that the device is connected to (e.g. waking up a home computer from a work network or the cellular provider's network). Please see this DSLReports.com article for information about configuring your router.
Host Name
Enter the domain name or IP address where the magic packet should be sent.
Port
Enter the port that the magic packet will be sent to. By default the packet will be sent to port number 9. Port number should be numeric.
